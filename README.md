## Установка зависимостей проекта:
    ```bash
    make install
    ```
## Проверка линтера:
    ```bash
    make lint
    ```

## Проверка линтера и исправление ошибок:
    ```bash
    make lint-fix
    ```

## Войти в БД для просмотра комплектующих и сборок:
    ```bash
    make database
    ```
