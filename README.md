# PCAMKR | ВЕБ-ПРИЛОЖЕНИЕ ДЛЯ АВТОМАТИЗАЦИИ ПОДБОРА КОМПЬЮТЕРНЫХ КОМПЛЕКТУЮЩИХ

Данное веб-приложение автоматизирует процесс выбора и сборки персонального компьютера. Оно помогает пользователям подобрать совместимые компоненты, рассчитать итоговую стоимость и проверить сборку на конфликты, даже если у них нет глубоких технических знаний.

## Цель проекта и его задачи:
 - Упростить выбор ПК для игр, работы и других задач
 - Исключить ошибки при подборе несовместимых деталей (например, процессора и материнской платы)
 - Автоматически рассчитывать потребляемую мощность и общую цену сборки.

### Общие команды:

1. Установка зависимостей:
    ```bash
    make install
    ```

2. Проверка линтера:
    ```bash
    make lint
    ```

3. Проверка линтера и исправление ошибок:
    ```bash
    make lint-fix
    ```

4. Войти в БД для просмотра комплектующих и сборок:
    ```bash
    make database
    ```
