# PCAMKR | ВЕБ-ПРИЛОЖЕНИЕ ДЛЯ АВТОМАТИЗАЦИИ ПОДБОРА КОМПЬЮТЕРНЫХ КОМПЛЕКТУЮЩИХ

Данное веб-приложение автоматизирует процесс выбора и сборки персонального компьютера. Оно помогает пользователям подобрать совместимые компоненты, рассчитать итоговую стоимость и проверить сборку на конфликты.

## Цель проекта и его задачи:
 - Упростить выбор ПК для игр, работы и других задач
 - Исключить ошибки при подборе несовместимых деталей (например, процессора и материнской платы)
 - Автоматически рассчитывать потребляемую мощность и общую цену сборки.

### Общие команды:

1. Установка зависимостей:
    ```bash
    make install
    ```

2. Проверка линтера:
    ```bash
    make lint
    ```

3. Проверка линтера и исправление ошибок:
    ```bash
    make lint-fix
    ```

4. Войти в БД для просмотра комплектующих и сборок:
    1. Заходим в postgreSQL ```bash
    make database
    ```
    2. Создаем пустую базу данных ```bash
    CREATE DATABASE pcamkr_db;
    ```
    3. Переходим в нашу пустую базу и восстанавливаем dump ```bash
    psql -U postgres -d pcamkr_db < pcamkr_dump.sql
    ```
    4. Проверяем работает ли база данных ```bash
    node backend/server.js
    ```
    и переходим по ссылке в браузере ```bash
    http://ТВОЙ_IP_адрес:3001/api/test
    ```
